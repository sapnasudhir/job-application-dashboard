Connect GitHub

To save an uploaded CSV back to the repo, paste a fine-grained GitHub personal access token scoped to just job-application-dashboard with Contents: Read and write permission.

Create a token
  1. Go to github.com/settings/tokens?type=beta
  2. Generate new token, restrict to repository job-application-dashboard
  3. Permissions → Contents → Read and write
  4. Copy the token and paste it above
